# hdd-cli
脚手架工具

# 全局安装
yarn global add hdd-cli

# 帮助
hdd-cli --help

Usage: hdd-cli [options]

Options:
  -v, --version, -V  查看脚手架版本
  init               初始化项目
  -h, --help         display help for command
  
# 初始化脚手架
hdd-cli init
